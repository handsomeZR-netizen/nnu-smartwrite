## 主要更新内容

### 1. 品牌定位调整

### 2. 新增功能：全文评价

#### 功能描述
在评估表单中新增"AI 智能评价全文"按钮，用户可以：
- 评估选中的句子（原有功能）
- 评估整篇文章（新增功能）

#### 按钮样式
- **评估选中句子**: 紫粉红渐变，霓虹发光效果
- **评价全文**: 绿色渐变（NNU 品牌色）

#### 使用场景
- 学生完成整篇作文后，可以一键获取全文评价
- 教师可以快速评估学生的整体写作水平
- 支持多维度分析（词汇、逻辑、结构、内容）

### 4. AI 评估逻辑

#### 默认评估类型
- **保持不变**: 系统默认为写作评估
- 只有明确包含 "translate" 等关键词时才识别为翻译题
- 符合新的产品定位（写作辅导与评价平台）

#### 评估维度
- **写作题**: 词汇 (Vocabulary)、逻辑 (Logic)、结构 (Structure)、内容 (Content)
- **翻译题**: 信 (Faithfulness)、达 (Expressiveness)、雅 (Elegance)、语法 (Grammar)

#### 前端优化
- 新增全文评价按钮的事件处理
- 优化按钮布局和样式

#### 数据结构

- 更新 `directions` 字段以反映写作任务

## 使用指南

### 评估选中句子
1. 在"题目要求"中输入写作任务
2. 在"文章全文"中粘贴文章
3. 用鼠标选中需要评估的句子
4. 点击"AI 智能评估选中句子"按钮

### 评价全文
1. 在"题目要求"中输入写作任务
2. 在"文章全文"中粘贴完整文章
3. 直接点击"AI 智能评价全文"按钮
4. 系统将评估整篇文章的质量

## 部署说明

### 本地开发
```bash
npm run dev
```

### 生产构建
```bash
npm run build
npm start
```

### 环境变量
确保 `.env.local` 中配置了 DeepSeek API 密钥：
```
DEEPSEEK_API_KEY=your_api_key_here
```
