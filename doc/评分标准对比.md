# 评分标准对比：考研 vs 四六级

## 📊 核心差异对比表

| 维度 | 考研标准 | 四六级标准 |
|------|---------|-----------|
| **目标人群** | 研究生入学考试 | 本科生英语水平测试 |
| **评分重点** | 思想深度、批判性思维 | 语言规范、句式多样 |
| **词汇要求** | 学术词汇、专业表达 | 高级词汇替换（think→maintain） |
| **句式要求** | 论证逻辑、段落展开 | 简单句→从句、非谓语 |
| **容错率** | 允许少量语法错误 | 零容忍基础语法错误 |
| **扣分重点** | 论证不充分、逻辑缺失 | 中式英语、词汇重复 |

## 🎯 评估维度对比

### 写作题雷达图

#### 考研标准（旧）
```
词汇 (Vocabulary)  - 词汇丰富度和准确性
逻辑 (Logic)       - 逻辑连贯性
结构 (Structure)   - 句式/篇章结构
内容 (Content)     - 内容深度和相关性
```

#### 四六级标准（新）
```
切题 (Relevance)   - 内容完整性与切题度 ⭐
丰富 (Variety)     - 词汇高级度与句式多样性 ⭐
连贯 (Coherence)   - 逻辑衔接与过渡词
规范 (Accuracy)    - 语法准确与拼写规范 ⭐
```

### 翻译题雷达图

#### 考研标准（旧）
```
信 (Faithfulness)    - 准确还原原文
达 (Expressiveness)  - 表达流畅自然
雅 (Elegance)        - 文体优美得体
语法 (Grammar)       - 语法正确性
```

#### 四六级标准（新）
```
准确 (Accuracy)    - 准确还原原文含义
通顺 (Fluency)     - 表达流畅自然
词汇 (Vocabulary)  - 词汇选择恰当 ⭐
句法 (Syntax)      - 句式结构正确 ⭐
```

## 💬 评估反馈对比

### 示例句子
```
I think learning English is very good.
```

### 考研标准评估（旧）

```
评分: B
分析: 句子表达了学习英语的重要性，但论证不够充分。
建议增加具体论据支持观点，深化论述。

优点:
- 观点明确
- 语法基本正确

不足:
- 缺乏深度论证
- 论据不够充分

润色建议:
Learning English is crucial for academic and professional development, 
as it provides access to global knowledge and enhances career prospects.
```

### 四六级标准评估（新）

```
评分: B (7-9分)
分析: 该句子基本切题但表达过于简单，词汇重复且句式单一，
属于典型的初中英语水平。虽然语义正确，但缺乏四六级考试
要求的语言丰富性和表达深度。

优点:
- 句子结构完整，主谓宾清晰
- 内容切合题目要求

不足:
- 词汇重复且低幼，连续使用'think'和'good'等基础词汇 ⭐
- 句式过于简单，都是SVO结构，没有使用任何从句 ⭐

润色建议:
It is widely acknowledged that acquiring proficiency in English is 
of paramount importance, as it significantly enhances one's 
employability in today's globalized job market, a perspective 
shared by the majority of professionals.
```

## 🔍 关键差异分析

### 1. 词汇评估

**考研标准：**
- 关注学术词汇的使用
- 强调专业性和准确性
- 例：使用 "crucial", "enhance", "prospects"

**四六级标准：**
- 明确指出"低幼词汇" ⭐
- 提供具体替换建议 ⭐
- 例：think → maintain/argue, good → positive/beneficial

### 2. 句式评估

**考研标准：**
- 关注段落逻辑展开
- 强调论证结构
- 例：增加论据、深化观点

**四六级标准：**
- 明确指出"简单句问题" ⭐
- 展示如何改写为从句 ⭐
- 例：SVO → 定语从句、状语从句、非谓语

### 3. 错误容忍度

**考研标准：**
- 允许少量语法错误（如果不影响理解）
- 重点看思想深度

**四六级标准：**
- 零容忍基础语法错误 ⭐
- 严格检查：主谓一致、时态、冠词、单复数
- 例：people's life → people's lives

### 4. 中式英语

**考研标准：**
- 提及但不作为主要扣分项

**四六级标准：**
- 严格指出并扣分 ⭐
- 例：do exercise → take exercise/work out

## 📈 评分分布对比

### 考研标准（旧）

```
S (95-100): 语义准确、语法完美、表达精妙
A (85-94):  语义正确、1-2处可改进、无语法错误
B (70-84):  部分正确、有明显问题、1-2处语法错误
C (<70):    严重语义或语法错误
```

### 四六级标准（新）

```
S (13-15分): 切题、表达清楚、连贯性好、基本无语法错误、句型丰富
A (10-12分): 切题、表达较清楚、较连贯、有少量语言错误
B (7-9分):   基本切题、表达不够清晰、勉强连贯、语言错误较多
C (<7分):    条理不清、思路紊乱、语言支离破碎、严重错误多
```

## 🎓 适用场景对比

### 考研标准适合：
- ✅ 研究生入学考试备考
- ✅ 学术论文写作
- ✅ 需要深度论证的场景
- ✅ 强调批判性思维

### 四六级标准适合：
- ✅ 本科生四六级备考 ⭐
- ✅ 基础英语能力提升 ⭐
- ✅ 词汇和句式训练 ⭐
- ✅ 避免中式英语 ⭐

## 💡 为什么选择四六级标准？

### 1. 用户群体匹配
- 本科生是主要用户
- 四六级是刚需考试
- 直击提分痛点

### 2. 评估更具体
- 明确指出哪些词太简单
- 具体展示如何改写句子
- 提供满分范文参考

### 3. 实用性更强
- 学生可以立即应用
- 评估结果直接用于备考
- 提分效果明显

### 4. 差异化竞争
- 市面上少有专门针对四六级的 AI 工具
- 评分标准专业、权威
- 符合官方阅卷标准

## 🔄 如何切换回考研标准？

如果需要切换回考研标准，可以：

1. 恢复 `lib/ai-prompt.ts` 中的 `BASE_ROLE`
2. 恢复原有的雷达图维度定义
3. 调整评估重点（从词汇句式 → 思想深度）

但建议：
- 保持四六级标准作为默认
- 可以添加"评估模式切换"功能
- 让用户选择：四六级模式 / 考研模式

## 📝 总结

四六级标准相比考研标准：

| 特点 | 优势 |
|------|------|
| **更具体** | 明确指出低幼词汇和简单句 |
| **更实用** | 直接用于四六级备考 |
| **更严格** | 零容忍基础语法错误 |
| **更贴合** | 符合本科生学习需求 |
