## 📊 核心改动总结

### 1. AI 角色转变

**修改前：**
```
南京师范大学英语写作与翻译评审专家（通用标准）
```

**修改后：**
```
四六级阅卷组长（CET Grader）
- 严格按照《全国大学英语四六级考试大纲》评分
- 采用总体印象评分法
- 重点关注语言规范性、连贯性和得体性
```

### 2. 评分标准映射

| 等级 | 分数 | 描述 |
|------|------|------|
| S | 13-15分 | 切题、表达清楚、连贯性好、基本无语法错误、句型丰富 |
| A | 10-12分 | 切题、表达较清楚、较连贯、有少量语言错误 |
| B | 7-9分 | 基本切题、表达不够清晰、勉强连贯、语言错误较多 |
| C | <7分 | 条理不清、思路紊乱、语言支离破碎、严重错误多 |

### 3. 雷达图维度调整

#### 写作题
```
旧: 词汇/逻辑/结构/内容
新: 切题/丰富/连贯/规范 ⭐
```

#### 翻译题
```
旧: 信/达/雅/语法
新: 准确/通顺/词汇/句法 ⭐
```

### 4. 评估重点转变

**从考研标准：**
- 思想深刻性
- 论证充分性
- 批判性思维

**到四六级标准：**
- ✅ 词汇升级（think → maintain）
- ✅ 句式多样（简单句 → 从句）
- ✅ 中式英语识别
- ✅ 基础语法零容忍
